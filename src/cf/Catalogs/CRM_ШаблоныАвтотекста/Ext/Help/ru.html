Справочник "Шаблоны автотекста"

Назначение

Справочник предназначен для определения автотекста, который используется при отправке электронных писем из встроенного почтового клиента. Он содержит список действий системы, описанных на языке 1С.

Поля:

  • Действие - указывается действие на встроенном языке 1С, которое необходимо будет выполнить при использовании автотекста;
  • Представление  - указывается, как будет отображаться текущий автотекст пользователю;
  • Назначение - указывается к какому объекту будет применяться соответствующий автотекст.
