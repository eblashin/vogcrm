Справочник "Настройки обмена с MS Exchange"

Назначение

Справочник предназначен для хранения настроек обмена с серверами MS Exchange.

Форма элемента содержит несколько страниц.

На первой настраивается подключение к серверу MS Exchange: указывается наименование настройки, выбирается подключение к серверу MS Exchange, учетная запись пользователя сервера MS Exchange (в настройках подключения и учетной запис и должен быть указан один и тот же сервер MS Excchange), а также часовой пояс (нужен для синхронизации времени с сервером MS Exchange, который всегда работает в нулевой часовой поясе GMT=0) и определяется использование данной настройки в регламентном задании.
