Справочник "Виджеты"

Назначение

Справочник используется для хранения виджетов подсистемы "Ключевые показатели".

Виджеты - это преднастроенные показатели. Панель виджетов отображается/скрывается по соответствующей команде меню кнопки "Ещё". Виджеты выводятся в форму АРМ "Мои дела" из списка доступных. Список виджетов можно открыть из контекстного меню панели виджетов (меню отображается по нажатию правой кнопки мыши). По нажатию на какой-либо виджет открывается гистограмма по выбранному показателю.
