Справочник "Договоры клиентов"

Назначение

Справочник предназначен для хранения договоров расчетов, заключенных с клиентами.

Данный справочник подчинен справочнику "Партнеры".

Список договоров с клиентом определяется теми договорами, которые подчинены соответствующему элементу справочника "Партнеры".

Для каждого клиента можно выбрать т.н. договор по умолчанию, который будет подставляться по умолчанию в документы.

Аналитический учет расчетов с клинтами ведется по каждому договору.
