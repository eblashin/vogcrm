Справочник "Состояния событий"

Назначение

Справочник предназначен для хранения состояний событий.

Событие может иметь несколько состояний:

запланировано - событие должно произойти в будущем;

в работе - событие происходит в данный момент;

отменено - событие не произошло; выбрав это состояние, пользователь может указать в поле Причина отказа причину отказа от проведения данного события, выбрав ее из справочника Причины отказов;

завершено - событие произошло; после установки этого состояния при записи документа система предлагает запланировать следующее событие по данному контрагенту.
