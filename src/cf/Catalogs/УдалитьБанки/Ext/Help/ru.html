Справочник "Банки"

В справочник "Банки" заносятся сведения о банках, в которых имеются расчетные счета как у собственных организаций, так и у контрагентов.

 

Справочник вызывается для ввода и редактирования информации из  раздела "Предприятие" / Справочники / "Банки".

 

Список банков можно заполнить автоматически, используя общероссийский классификатор банков. Заполнение производится по кнопке Добавить банки из классификатора РФ.

Справочник также можно обновить с диска "Информационно-технологическое сопровождение" или с сайта "РосБизнесКонсалтинг".

Справочник имеет многоуровневую, иерархическую структуру. Элементы справочника можно перегруппировать, перенося их из группы в группу.

 

В форме списка справочника доступна команда "Обновить банки из классификатора". Данная команда обновляет ранее выбранные (добавленные из классификатора) банки. Если для элемента банка включено ручное Изменение, обновления не произойдет.

В данный справочник заносится следующая информация:

  • Наименование банка. Наименование банка, в котором открыт банковский счет компании или контрагента.
  • БИК. Банковский идентификационный код.
  • Корр. счет. Корреспондентский счет.
  • Город. Данные о местонахождении банка.
  • Адрес. Индекс и почтовый адрес банка.
  • Телефоны. Телефоны для связи с работниками банка.
