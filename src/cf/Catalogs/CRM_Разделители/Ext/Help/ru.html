Справочник "Разделители этапов и групп товаров коммерческих предложений"

Назначение

Справочник предназначен для хранения списка Разделителей используемых для оформления табличной части документа "Коммерческое предложение".

Необходим для улучшения восприятия табличной части печатной формы коммерческого предложения.
Разделителем может быть название Этапа работ или Группа товаров указанных в коммерческом предложении.

Используется для заполнения строки в табличной части документа "Коммерческое предложение".

Если в справочнике у разделителя указана галочка Итог то в печатной форме по элементам входящим в раздел будет выводиться итог по разделу.
