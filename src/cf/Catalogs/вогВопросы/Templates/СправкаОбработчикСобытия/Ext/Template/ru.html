Доступны следующие переменные:

- ИдентификаторВопроса = МассивИмени[МассивИмени.ВГраница()];
- ИмяТаблицы = "Таблица_" + ИдентификаторВопроса;
- СхемаДанныхВопроса = СхемаДанныхВопросов[ИдентификаторВопроса]; 

- Объект - Текущий объект вогАнкета

- Элемент - текущий элемент формы

- ТекущиеДанные - Текущие данные строки

- А так же другие стандарные параметры

 

Описание оповещения:

Сигнатура процедуры:

&НаКлиенте
Процедура Подключаемый_ОписаниеОповещения(Значение, ДополнительноеЗначение) Экспорт

Используемые поля структуры дополнительных значений:

- КодВыполнения - выполняется данный код

- Параметр1, Параметр2, Параметр3 - Данные параметры устанавливаются перед выполнением кода в Выполнить.

Используется для вложенных обработчиков, тем самым создавая CallbackHell, но удобно ;)

| 
|КонецЕсли;
|";
