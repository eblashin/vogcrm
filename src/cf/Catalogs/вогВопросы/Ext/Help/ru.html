- Алгоритм обработки ответов:

Доступные переменные:

- Вопрос - Ссылка на элемент вопроса

- ТаблицаВопроса - Таблица значений

- ДанныеВариантовОтветов - Таблица значений

Выполняется в обработке проведения документа Анкета

Отбор по респонденту

Если  в параметрах СКД присутствует параметр "Респондент", тогда устанавливается отбор по этому полю.

Отбор по направлению деятельности

Если  в параметрах СКД присутствует параметр "НаправленияДеятельности", тогда устанавливается отбор по этому полю.

Значение данного поля берется из варианта опроса.

Предполагается, что если параметр пустой - то это обрабатывается условием запроса.
