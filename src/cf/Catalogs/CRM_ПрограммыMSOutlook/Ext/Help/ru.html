Справочник "Программы MS Outlook"

Назначение

Механизм "Обмен данными с MS Outlook" позволяет реализовать двухсторонний обмен данными между информационной базой 1С:CRM  и  установленной на компьютере пользователя программой MS Outlook.

Справочник хранит данные о всех программах MS Outlook с которыми выполняется обмен данными. Каждая программа MS Outlook привязана к пользователю и компьютеру. При открытие списка справочника в нем отображаются только программы по текущему пользователю/компьютеру. Для просмотра списка всех программ MS Outlook предназначена кнопка "Отображать все".

Синхронизация данных выполняется по настройкам пользователя. Для каждой программы  MS Outlook  может существовать несколько настроек обмена данными.

Перед первым обменом данными необходимо добавить новую настройку обмена.  Дополнительную информацию о технологии обмена  можно  найти на форме настройки обмена данными.

Реквизиты:

Дата последней синхронизации контактов - дата последнего обмена контактами между программами 1С:CRM и MS Outlook;
Дата последней синхронизации писем - дата последнего обмена электронными письмами (в том числе исходящими) между программами 1С:CRM и MS Outlook;
Дата последней синхронизации событий - дата последнего обмена событиями между программами 1С:CRM и MS Outlook;
Компьютер - компьютер, для которого используется текущая настройка обмена данными;
Пользователь - пользователь, для которого используется текущая настройка обмена данными.
