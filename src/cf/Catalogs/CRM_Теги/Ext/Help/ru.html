Справочник "Теги"

Назначение

Хранит теги. С целью ведения дополнительной аналитики для интересов клиентов реализован механизм тегов. Теги задаются внутри подразделения, для логической быстрой отметки, например, что клиент интересовался бассейнами и кондиционерами — в данном случае применено два тега.
Ведение аналитики с помощью тегов осуществляется в разрезе подразделений. Для каждого подразделения задаются такие теги, которые используются для маркировки документов продажи. Каждому подразделению необходимо вести свой набор тегов. При работе с Интересом пользователь может добавлять в него теги, но только из списка доступных для данного подразделения.По тегам возможны отборы интересов клиентов в Воронке продаж.
