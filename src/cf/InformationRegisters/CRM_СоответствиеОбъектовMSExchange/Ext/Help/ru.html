Регистр сведений "Соответствие объектов MS Exchange"

Назначение

Регистр сведений предназначен для хранения соответствий объектов 1С и MS Exchange.

Создается в подсистемах:

«Обмен с MS Exchange»

Измерения:

Сервер MS Exchange

Сервер MS Exchange. (Серверы MS Exchange)

Объект 1С

Объект из 1С (Любая ссылка)

Ресурсы:

ID

Уникальный идентификатор объекта из 1С (Строка)

Change key

Уникальный идентификатор объекта Exchange (Строка)

УчетнаяЗапись

Учетная запись пользователя Exchange (Учетные записи MS Exchange)

Дата последней модификации

Дата последней модификации объекта в MS Exchange (Дата)
