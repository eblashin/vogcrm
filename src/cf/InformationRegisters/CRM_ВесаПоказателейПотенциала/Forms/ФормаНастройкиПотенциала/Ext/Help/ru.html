Мастер установки показателей

В Мастере установки показателей необходимо задать формулу расчет потенциала клиента на основании реквизитов портрета клиента.

Для нужных реквизитов необходимо задать веса для каждого из его значений. Согласно введенной формуле, в зависимости от значения показателя для конкретного клиента, в алгоритм расчета подставляются конкретные веса выбранных реквизитов и рассчитывается итоговый потенциал.

Потенциалы клиентов рассчитываются автоматически регламентным заданием "Расчет потенциала клиентов».
