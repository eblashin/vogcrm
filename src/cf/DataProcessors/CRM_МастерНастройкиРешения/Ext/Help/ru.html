Мастер настройки решения 1C:CRM

Назначение

Мастер настройки решения предназначен для помощи по вводу сведений, необходимых для начала работы с программным продуктом 1C:CRM, а именно:

■      задание основных параметров решения;

■      заполнение сведений об организации.

 

Мастер настройки решения 1C:CRM разделен на две части:

■      Начало работы — позволяет указать минимальные настройки для начала работы в программе. Рекомендуется выполнять данную настройку при перво­начальном знакомстве с программой. Использование только этой части мастера позволяет сделать работу в программе наиболее простой и интуитивно понятной для пользователей. В первой части мастера настраиваются следующие параметры:

○     Сведения о компании;

○     Структура компании и права доступа;

○     Организация клиентской базы;

○     Оповещения о событиях.

■      Расширенные возможности — эту часть мастера рекомендуется настраивать уже после начала работы в решении, для выполнения более сложных задач. В этой части мастера содержатся следующие пункты (шаги настройки):

○     Управление клиентской базой;

○    Сегментирование;

○     Маркетинговые кампании;

○     Управление продажами;

○     Управление проектами/процессами;

○     Интеграция с другими системами;

○     Интеграция с телефонией.

 

Процесс ввода данных с помощью Мастера настройки решения представляет собой пошаговое заполнение информации в диалоговых окнах, которые отображаются на экране. Можно последовательно выполнить шаги по настройке параметров или же открыть формы для каждого шага настройки отдельно.
