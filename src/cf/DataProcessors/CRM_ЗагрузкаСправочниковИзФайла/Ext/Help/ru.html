Обработки "Загрузка справочников из файла"

Назначение

Обработка предназначена для первоначальной, а также регулярной загрузки элементов в справочники "Клиенты" и "Товары/услуги".

ФОРМАТЫ ФАЙЛОВ ДЛЯ ЗАГРУЗКИ ДАННЫХ ИЗ ФАЙЛА

 

С помощью обработки возможно загружать данные из следующих форматов файлов:

  • Файлы выгрузки контактов из программ "MS Outlook", "MS Outlook Express", "The Bat!";
  • Файлы в формате MS Excell, в которых в первой строке начала файла, расположены названия колонок, а под каждым названием колонок располагаются значения по этой колонке. Каждая строка такого файла (кроме первой с названиям колонок) считается программой одним элементом справочника, колонки в котором являются реквизитами элемента справочника;
    Примечание:
    Для правильной работы загрузки в WEB-клиенте выполните следующие действия:
    В настройках IE на закладке "Безопасность" выберите зону настройки параметров безопасности "Местная интрасеть" и измените "Уровень безопасности для этой зоны" нажав кнопку "Другой".
    В открывшемся окне "Параметры безопасности - зона местной интрасети" включите все элементы в группе "Элементы ActiveX и модули подключения"
  • Файлы в форматах ".txt" с разделителями (например ";"), где в первой строке указываются названия колонок через разделитель. Каждая строка такого файла (кроме первой с названиям колонок) считается программой одним элементом справочника, колонки в котором являются реквизитами элемента справочника.

 

ПОРЯДОК ЗАГРУЗКИ ДАННЫХ ИЗ ФАЙЛОВ 

 

Для загрузки данных из указанных форматов необходимо выполнить следующие действия:

  • В реквизите "Источник" выбрать нужный формат файла для загрузки данных. Этот формат должен соответствовать формату загружаемого файла.
  • Выбрать файл для загрузки в реквизите "Файл данных". При выборе файла данных программа анализирует данные из файла по выбранному формату.
  • Выбрать (при необходимости) папку в справочнике, куда будут загружены новые элементы справочника.
  • В табличной части "Сопоставление полей справочника с полями файла" выбрать для реквизитов справочника (в левой части таблицы) поля, которые соответствуют этим реквизитам в файле. Для этого нажмите правой кнопкой мышки или клавишей Ввод в правой части таблицы напротив нужного реквизита строчку. Откроется список доступных полей (например, для файлов в формате .xls полем будет являться колонка в первой строке) и выберем нужное поле. При выборе команды подбор несколько полей можно сопоставить для одного реквизита справочника несколько полей из файла.
  • Сопоставить, как  в пункте выше все необходимые реквизиты. Причем если для реквизита справочника не будет сопоставлено поле из файла - данные в этом реквизит загружаться не будут.
  • Если требуется сопоставить поле даты, то следует учитывать следующее: для файлов .xls дата должна быть в формате 'дд.мм.гггг'.
  • После всех сопоставлений нажать кнопку "Загрузить" и после этого программа произведет загрузку элементов справочника в информационную базу.

 

ПРАВИЛА СИНХРОНИЗАЦИИ СПРАВОЧНИКОВ ПРИ ЗАГРУЗКЕ

 

Обработка может использоваться, как для первоначальной загрузки справочников в информационную базу (ввод начальных данных), так и для обновления данных в справочниках. Например, таким образом можно подгружать прайс-лист из внешнего файла и обновлять цены по товаров/услугам.

При регулярной загрузке данных в справочник поиск, на уже введенные элементы, происходит по полному наименованию реквизитов, выделенных жирным шрифтом. Например, для справочника "Товары/услуги" таким реквизитом является "Наименование".
