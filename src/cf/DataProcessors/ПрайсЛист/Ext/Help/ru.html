Обработки "Прайс-лист"

Назначение

Обработка Прайс-лист предназначена для просмотра и редактирования цен на товары компании.

 

Нажатие кнопки Отправить по почте позволяет отправить данный прайс-лист по электронной почте.
При этом пользователь может выбрать, в каком формате следует добавлять в письмо файл прайс-листа и следует ли архивировать этот файл. Может быть выбрано несколько форматов сразу.
