Обработка "Передача текущих дел"

Обработка "Передача текущих дел" предназначена для изменения ответственного лица у объектов (потребность может возникнуть, например, при передаче дел увольняемого сотрудника).

В форме обработки следует указать Ответственного и Нового ответственного (Справочник "Пользователи").

В поле отбора следует выбрать те объекты, для которых будет переназначено ответственное лицо.

По кнопке "Далее" будут отобраны указанные объекты за период. Можно выбрать нужные объекты и по кнопке "Выполнить" передать ответственность указанному пользователю.

В случае, когда в качестве объекта передачи указаны задачи, также будут изменены настройки исполнителей для карт маршрутов и бизнес-процессов.

Это требуется для того, чтобы при дальнейшем прохождении бизнес-процесса или при создании нового созадвались задачи на нового ответственного.
