Типы  данных

Календарь

Синхронизируются документы «Мероприятия» информационной базы 1C:CRM  и события календаря MS Outlook.

Исключения

Не синхронизируются события MS Outlook с установленным признаком "повторяющееся" .

Почта

Синхронизируются документы «Событие» информационной базы 1C:CRM  и почтовые сообщения программы MS Outlook.

Доступа только загрузка почтовых сообщений из программы MS Outlook в программу 1C:CRM!

Контакты

Синхронизируются элементы справочников Контрагенты, Контактные лица контрагентов, Контактные лица и Личные контакты   со списком контактов  MS Outlook. 

Правила установки соответствия типов данных  

Контакт, загружаемый из MS Outlook, будет загружен в соответствующий справочник 1С:CRM исходя из следующих правил:

Заполнены поля контакта MS Outlook НЕ заполнены поля контакта MS Outlook Признак "Частное" контакта MS Outlook Тип данных 1С
Организация Фамилия, Имя, Отчество Любое значение Справочник Контрагенты
Организация, Фамилия, Имя, Отчество Любое значение Справочник Контактные лица контрагента
Фамилия, Имя, Отчество Организация Не установлен Справочник Контактные лица
Фамилия, Имя, Отчество Организация Установлен Справочник Личные контакты

Если заполнено только наименование организации тогда контакт загрузится в справочник Контрагенты. Если заполнено наименование организации и заполнено хотя бы одно из полей Имя, Отчество или Фамилия; тогда контакт загружается в справочник Контактные лица контрагентов. Если наименование организации не заполнено тогда частные контакты загружаются в справочник "Личные контакты", не частные контакты загружаются в справочник "Контактные лица".

Особенности загрузки новых элементов в справочник Контактные лица контрагента

При загрузки новых контактов в справочник Контактные лица контрагента из MS Outlook будет выполнен поиск контрагента (Справочник Контрагенты) и контактного лица (Справочник Контактные лица). Если контрагент или контактное лицо не найдено тогда будут созданны новые элементы. Поиск контрагента выполняется по наименованию (реквизит организация контакта MS Outlook), поиск контактного лица выполняется по реквизитам фамилия, имя и отчество.  

Исключения:

Элементы справочника Контактные лица не выгружаются в программу MS Outlook.
