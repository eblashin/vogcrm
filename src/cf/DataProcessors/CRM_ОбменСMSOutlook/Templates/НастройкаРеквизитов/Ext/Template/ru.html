Синхронизация участников мероприятия календаря MS Outlook

В мероприятие календаря MS Outlook список участников может быть выбран в поле "Контакты" или "Участники". Для синхронизации данных полей с участниками документа "Мероприятие" (только контрагенты и контактные лица) необходимо установить настройки "Синхронизировать поле контакты мероприятия MS Outlook" и "Синхронизировать участников мероприятия MS Outlook".   

Новые участники мероприятия будут созданы в базе приемнике  в независимости от того установлен или нет признак синхронизации контактов.

Особенности синхронизации

Если в документе Мероприятие начало события является началом дня, а окончание мероприятия является окончанием дня, тогда мероприятиезагружается в MS Outlook с признаком "Мероприятие на целый день".

Значения реквизитов Территория, Помещение переносятся только при синхронизации в направление синхронизации 1С:CRM->MS Outlook.

Особенности синхронизации реквизитов Контрагент и Контактное лицо справочника Контактные лица контрагента

Реквизиты Контрагент и Контактное лицо являются реквизитами обязательными для заполнения.  При загрузки новых контактов в справочник Контактные лица контрагента выполняется  поиск контрагента (Справочник Контрагенты) и контактного лица (Справочник Контактные лица). Если контрагент или контактное лицо не найдено тогда в соотв. справочники будут добавлены новые элементы.  Поиск контрагента выполняется по наименованию (реквизит организация контакта MS Outlook), поиск контактного лица выполняется по реквизитам фамилия, имя и отчество.

Синхронизация поля "Заметки" контакта MS Outlook

По умолчанию поле "Заметки" контатка в программе MS Outlook синхронизируется с соот. полем контакта в программе 1С. При необходимости можно отключить синхронизацию значений данного поля.

Синхронизация контактной информации

Структура хранения контактной информации в информационной базе 1С и программе MS Outlook различна. Настройка соответствие видов контактной информации выполняется в специальном диалоге. В форме настройки необходимо установить соответсвие для всех используемых видов контактной информации. 

Синхронизируются только сопоставленные виды контаткой информации!

Контактная информация каждого вида загружается в базу 1С:CRM из контакта MS Outlook только в случае ее отсутвтия в контакте 1С. Например если для контакта MS Outlook  был добавлен или изменен номер домашнего телефона, тогда данный номер будет загружен в базу 1С только в случае если у контакта 1С номер домашнего телефона не был введен до синхронизации.

Значения реквизитов "по умолчанию" для новых документов Событие загружаемых из MS Outlook

Пользователь может указать  значения  "по умолчанию" для реквизитов  "Группа события",  "Вид события" и  "Состояние события".  

Тип события (входящее/исходящее событие) определяется автоматически исходя из направления письма MS Outlook.

Синхронизация участников письма MS Outlook

Новые участники события будут созданы в базе приемнике  в независимости от того установлен или нет признак синхронизации контактов.

Синхронизация вложений письма MS Outlook

Вложения письма MS Outlook загружаются в 1С:CRM как прикрепленные файлы.
