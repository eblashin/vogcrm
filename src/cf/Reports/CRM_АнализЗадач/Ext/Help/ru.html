Отчет "Анализ задач"

Назначение:

Отчет предназначен для получения статистических данных по выполнению задач.
 
В сформированном отчете можно увидеть общее количество задач, какие задачи были выполнены, переадресованы. Также отображена длительность выполнения поручений в часах и днях.
Для получения детальной информации по задачам можно выбрать для формирования другие варианты данного отчета.

«Бизнес-процессы»
