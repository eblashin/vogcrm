Отчет "Мой автомотиватор"

Назначение

Отчет «Мой автомотиватор» позволяет оценить планомерность загрузки менеджера по его планированию взаимодействий с клиентами. Отчет состоит из трех частей:
• график загруженности менеджера в баллах;
• таблица по дням с загрузкой менеджера по видам взаимодействий и их количеству, количество баллов;
• таблица фактических документов по коммуникациям (Телефонные звонки, Электронные письма, Встречи с клиентами и др.).


Отчет «Мой автомотиватор» менеджеру по продажам необходимо использовать для равномерного распределения загрузки, в зависимости от вида планируемого взаимодействия.
