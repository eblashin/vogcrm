Отчет "Контроль руководителя"

 

 

Отчет отображает анализ по клиентам и их интересам.

 

В отчете используются три предопределенных варианта отчета:

- Клиенты (отображаются все клиенты и потенциальные клиенты с данными по их интересам, если такие есть)

- Интересы (отображаются все интересы с данными по ним)

- Клиенты и интересы (совмещает два предыдущих варианта)

 

В отчете используется цветовая индикация заполненности и актуальности данных - первая колонка отчета имеет три варианта цветовой индикации (зеленый, желтый, красный).

 

Для варианта "Клиенты":

Зеленый - вся обязательная контактная информация по контактным лицами клиента (либо по потенциальному клиенту) заполнена, портрет клиента заполнен и по клиенту/потенциальному клиенту есть активный интерес.

Желтый - не заполнена совсем или частично контактная информация по контактным лицами клиента (либо по потенциальному клиенту), или по портрету клиента не заполнена вся информация.

Красный - не заполнен портрет клиента и не заполнена контактная информация по контактным лицами клиента (либо по потенциальному клиенту), или нет активного интереса по клиенту (когда клиент имеет статус “Работаем”).

 

Для варианта "Интересы ":

Зеленый - по интересу дата следующего взаимодействия не просрочена.

Желтый - интерес не в состоянии “Первичный интерес” и просрочено следующее взаимодействие по нему.

Красный - интерес в состоянии “Первичный интерес” и просрочено следующее взаимодействие по нему.

 

Для варианта "Клиенты и интересы":

Зеленый - вся обязательная контактная информация по контактным лицами клиента (либо по потенциальному клиенту) заполнена, портрет клиента заполнен и по клиенту/потенциальному клиенту есть активный интерес, а также имеется интерес по данному клиенту и дата следующего взаимодействия по нему не просрочена.

Желтый - выполняется любое из двух условий: 1. Не заполнена совсем или частично контактная информация по контактным лицами клиента (либо по потенциальному клиенту), или по портрету клиента не заполнена вся информация; 2. Имеется интерес по клиенту, интерес не в состоянии “Первичный интерес” и просрочено следующее взаимодействие по нему.

Красный - выполняется любое из двух условий: 1. Не заполнен портрет клиента и не заполнена контактная информация по контактным лицами клиента (либо по потенциальному клиенту), или нет активного интереса по клиенту (когда клиент имеет статус “Работаем”); 2. Имеется интерес по клиенту, интерес в состоянии “Первичный интерес” и просрочено следующее взаимодействие по нему.
