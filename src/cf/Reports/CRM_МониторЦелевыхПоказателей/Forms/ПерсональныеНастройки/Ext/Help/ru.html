Какие способы группировки можно установить в мониторе целевых показателей?

Можно установить один их следующих вариантов группировки: по важности и по категориям целей предприятия.

По важности. Все показатели будут разбиты на следующие группы в соответствии со значением статуса и целевого тренда показателя.

  • Критическое состояние.
  • Важно
  • К сведению
  • Данные неактуальны
  • Ошибка расчета
  • Пустой результат расчета

По категориям целей. Все показатели в соответствии с реквизитом Категория целей справочника Структура целей.

Какие варианты отображения показателя можно применять в мониторе?

Пользователь может выбрать один из двух вариантов краткого представления показателей.

  • Только значение. Будет показано только значение показателя.
  • Значение и детали. Кроме значения показателя будут показаны детальные данные: об относительном и абсолютном изменении, сравнение с прошлым измеренным значением и т.д.

Можно ли изменить количество колонок, в которых представлены данные в мониторе?

По умолчанию данные в мониторе выводятся в три колонки. При необходимости пользователь может установить любое количество колонок.

Можно ли увеличить размер диаграмм или таблиц, представленных в мониторе?

По умолчанию в мониторе установлен малый размер диаграмм и таблиц. При изменении значения на крупный, размер по вертикали будет увеличен в два раза.

Какой период обновления данных установлен для монитора?

По умолчанию период обновления установлен, равным 5 минутам. Пользователь может установить любую длительность (в минутах) периода автоматического пересчета данных в мониторе показателей. При этом будут обновляться только те показатели, период актуальности которых истек.

Как изменить данные по выводимым вариантам анализа в мониторе?

Настройка списка вариантов анализа, которые будут выводиться в монитор целевых показателей, определяется на странице "Варианты анализа".

По умолчанию все доступные пользователю варианты анализа выводятся в виде диаграмм. Доступность того или иного варианта анализа определяется при настройке варианта анализа.

  • Видимость варианта анализа в мониторе.
  • Вариант отображения. Доступные варианты: кратко, диаграмма, таблица.
  • Вывод деталей для краткого представления и таблиц.
  • Вывод итогов для таблиц.
