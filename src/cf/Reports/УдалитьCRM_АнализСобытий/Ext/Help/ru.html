Отчет "Анализ событий"

Назначение

Для отчета предусмотрены различные варианты формирования.

 

Отчет "Анализ событий (диаграмма)" отображает события, зарегистрированные в системе в текущем месяце. Отчет формируется с разбивкой по ответственным менеджерам.

Отчет "Анализ событий (Запланированные события по ответственным за текущий месяц (диаграмма))" позволяет отобразить запланированные дела по менеджерам. Отчет формируется с разбивкой по ответственным менеджерам. С помощью данного отчета можно проанализировать незавершенные дела менеджеров.

Пользователь может указать тип диаграммы, задать отбор по определенным критериям.

Отчет "Анализ событий (интересы клиентов)" отображает интересы клиентов по товарам и услугам, зарегистрированным в информационной базе.

Отчет "Анализ событий (интересы клиентов по содержанию без указания товаров)" отображает интересы клиентов по товарам и услугам, не зарегистрированным в информационной базе (указано содержание товара/услуги).
