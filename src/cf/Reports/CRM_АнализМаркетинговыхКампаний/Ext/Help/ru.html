Отчет "Анализ маркетинговых кампаний"

Назначение

Отчет отображает плановые параметры маркетинговых кампаний, которые определяются отделом маркетинга.
Данные для отчета формируются по следующим показателям: количество маркетинговых мероприятий, плановая сумма продаж, количество новых привлеченных клиентов, плановая сумма затрат на данные мероприятия.
