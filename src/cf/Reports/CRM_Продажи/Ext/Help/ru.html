Отчет "Продажи"

Назначение

Отчет предназначен для получения всей информации по проведенным продажам товаров/услуг клиентам в информационной базе. Имеются преднастроенные варианты отчета, которые позволяют просмотреть продажи в разрезах:

    • По дням/неделям.
    • По клиентам.
    • По менеджерам.
    • По регионам и клиентам.
    • По товарам/услугам.
    • По регионам и товарам.
    • Оценка потенциала продаж.
