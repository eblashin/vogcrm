Отчет "Анализ сегментов клиентов"

Назначение:

Есть два варианта формирования отчета "Анализ сегментов клиентов".

  • Анализ сегментов клиентов (по клиентам);
  • Анализ сегментов клиентов (по ответственным менеджерам).

 

Отчет "Анализ сегментов клиентов (по клиентам)" отражает динамику взаимодействий с клиентами. Данные формируются по сегментам, за прошлый и текущий месяцы.
При помощи данного отчета пользователь может проанализировать активность взаимодействий с клиентами и, при необходимости, предпринять действия по улучшению ситуации.

Отчет "Анализ сегментов клиентов (по ответственным менеджерам)" отражает динамику взаимодействий с клиентами. Данные формируются по менеджерам, за прошлый и текущий месяцы.
При помощи данного отчета пользователь может проанализировать активность менеджеров по взаимодействию с клиентами.
