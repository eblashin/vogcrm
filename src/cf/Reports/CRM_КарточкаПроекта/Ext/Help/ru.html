Отчет "Карточка проекта"

Назначение

Отчет служит для просмотра контрольных точек. Также при помощи данного отчета можно увидеть текущие задачи, бизнес-процессы по проекту, которые запущены или еще не выполнены. 

Также из карточки проекта можно сформировать отчет по план-факту рабочего времени и по плановым затратам проекта.
