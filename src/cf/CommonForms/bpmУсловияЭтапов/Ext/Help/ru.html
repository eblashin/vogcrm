Условие маршрутизации - это набор правил, по которым проверяются условия на выполнение и в случае упесшной проверки - выполняется назначенное действие. Например, при создании документа "Коммерческое предложение" на сумму свыше 50 000 руб. выполняется оповещение Руководителю отдела продаж по электронной почте. 

Есть три способа создания условий:

  1. В режиме конструктора - укажите одно или несколько правил сравнения и сгруппируйте их по принципу И, Или или Не. Выберите реквизиты, вид сравнения и значение, с которым нужно сравнивать. Например, чтобы отправлять оповещение только для клиента у которого заполнен тип отношений "Покупатель".

Цель - Равно - Email-оповещение, 

Клиент.Покупатель - равно - "ДА".

  1. На встроенном языке "1С" - впишите код условия с указанием значения переменной Результат типа Булево (Ложь или Истина). Если это значение не присвоено, то условие выполняться не будет. Например, чтобы проверить обязательное заполнение поля клиента "ИНН", впишите следующее выражение:

Если ЗначениеЗаполнено(Предмет.CRM_Партнер) И ЗначениеЗаполнено(Предмет.CRM_Партнер.ИНН) Тогда
	Результат = Истина;
Иначе      Результат = Ложь;
КонецЕсли;
Комбинация из других условий - подберите из справочника уже созданные ранее условия и свяжите их в логическую комбинацию с помощью группировки.
