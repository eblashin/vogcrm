Общая форма "Персональные настройки"

На форме "Персональные настройки" задаются значения настроек и параметров, которые применяются для текущего пользователя системы и определяют используемый текущим пользователем функционал программы.

Закладка «Настройки»

На закладке устанавливаются общие настройки текущего пользователя.

 

Параметры использования CRM:

Используемый вариант поставки - выбирается вариант ПРОФ или КОРП. Настройка позволяет распределить лицензии по пользователям, когда совместно используются лицензии и ПРОФ и КОРП.
Данная настройка имеет больший приоритет, чем параметр в настройках лицензирования.

 

Основные значения для подстановки в документы и справочники:

Основная организационно-правовая форма клиента - выбирается основная организационно-правовая форма (ООО, АО и т.д.). автоматически подставляемая при вводе нового клиента.

Основная организация - выбирается основная организация для заполнения в новых документах.

Основной вид цен продажи - выбирается основной вид цен, который будет использоваться при продаже.

Основной ответственный - ответственный, который будет подставляться во все документы, при создании.

Основной проект - проект, который будет подставляться во все документы при создании.

Категория события по умолчанию - категория, которая по умолчанию будет подставляться в событие при создании.

 

Настройки календаря:

Указываются настройки времени начала и окончания рабочего дня. Имеют больший приоритет, чем значения в Общих настройках системы.

 

Общие настройки:

Задавать вопрос при заполнении события по шаблону из проекта - Используется при выборе Проекта в событии,  когда в Проекте указан шаблон события.

Использовать календарь в оповещениях - определяет, будет ли учитываться основной производственный календарь при формировании оповещений.

Использовать помощник ввода нового клиента - включает использование мастера быстрого ввода клиента.

Автоматически связывать взаимодействия с бизнес-процессами - при записи взаимодействий (письма, события и т.д.) произойдет автоматическое связывание с активным бизнес-процессом (если он есть).

Подставлять основной проект в поручения - указывает, нужно ли автоматически подставлять Основной проект при вводе Поручения.

 

Настройки календаря Google:

Синхронизировать календарь с Google - данная настройка позволяет пользователю использовать синхронизацию событий с календарем Google. Предварительно должна быть включена глобальная настройка в Общих настройках системы.

Календарь Google - календарь Google, с которым будет производиться синхронизация.

 

СофтФон:

Использовать СофтФон - включает возможность использования СофтФон для интеграции с различными телефонными системами. Предварительно должна быть включена глобальная настройка в Общих настройках системы.

Использовать внутреннюю панель СофтФон (1С) - разрешает использовать внутреннюю панель СофтФон вместо внешней (системной). Рекомендуется использовать внешнюю панель СофтФон.

Настройка автоматического создания документов (только для внешних звонков) - включает автоматическое создание События при "снятии трубки" для входящих и/или исходящих звонков.

 

Закладка «Электронная почта»

На закладке устанавливаются настройки электронной почты для текущего пользователя: Основная учетная запись и подписи писем.

В поле  "Основная учетная запись" можно выбрать только те учетные записи, к которым есть доступ у пользователя. 

Подписи для исходящих и пересылаемых писем задаются для каждой учетной записи отдельно.

Закладка «Оповещения»

На закладке устанавливаются настройки оповещений для текущего пользователя.

 

Оповещения о значимых событиях:

Оповещать о действиях по клиентам - при создании нового документа, где выбран клиент, соответствующий отбору (отбор настраивается в колонке "Отбор" и настраивается по различным параметрам), будет создано оповещение (SMS, E-mail или Напоминание).

Оповещать о действиях по проектам - при создании нового документа, где выбран проект, соответствующий отбору (отбор настраивается в колонке "Отбор" и настраивается по различным параметрам), будет создано оповещение (SMS, E-mail или Напоминание).

Оповещать о Коммерческом предложении с суммой более - если сумма "Коммерческого предложения" будет превышать сумму, указанную в колонке "Отбор", то будет создано оповещение (SMS, E-mail или Напоминание).

Оповещать о Счете на оплату с суммой более - если сумма "Счета на оплату" будет превышать сумму, указанную в колонке "Отбор", то будет создано оповещение (SMS, E-mail или Напоминание).

Оповещать о завершении бизнес-процессов - после завершения бизнес-процесса будет создано оповещение (SMS, E-mail или Напоминание).

Оповещать о просроченных контрольных точках - когда контрольная точка будет просрочена, будет создано оповещение (SMS, E-mail или Напоминание).

Оповещать о просроченных событиях/задачах - когда событие/задача будет просрочено, будет создано оповещение (SMS, E-mail или Напоминание).

 

Оповещать о событиях/задачах:

Оповещать о новых задачах - при создании новой задачи будет создано оповещение (SMS, E-mail или Напоминание).

Оповещать о новых входящих письмах - при получении нового входящего письма будет создано Напоминание со ссылкой на письмо. Если писем будет большое количество, будет создано одно общее Напоминание.

Оповещать о переадресованных документах и задачах - при переадресации документов или задач будет создано оповещение (SMS, E-mail или Напоминание).

 

Оповещения о запланированных событиях:

Указываются настройки оповещений (SMS, E-mail или Напоминание) с выбором Срока и Периода оповещения, которые автоматически будут указываться при создании новых событий на закладке "Оповещения".

В форме События можно добавить вручную более 3 оповещений.

 

Оповещения о днях рождения:

Указываются настройки оповещений (SMS, E-mail или Напоминание) с выбором Срока и Периода оповещения, которые будут использоваться при формировании оповещений о днях рождения Клиентов и Контактных лиц.

 

Закладка «Цветовые категории задач»

На закладке вводятся персональные категории задач с возможность указания цвета категории.

 

Закладка «Избранные получатели»

На закладке устанавливаются избранные получатели для текущего пользователя.

Пользователи, добавленные в данный список, будут использоваться для быстрой переадресации по кнопке [ Перенаправить]

 

Закладка «Работа с файлами»

На закладке устанавливаются персональные настройки работы с файлами текущего пользователя.

 

Закладка «Прочее»

На закладке устанавливаются сведения о пользователе и вывод запроса на подтверждение при завершении работы.
