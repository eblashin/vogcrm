Настройка этапов бизнес-процессов

 

Данная форма служит для настройки этапов выбранной карты маршрута.

В левой части находится поле отображения карты маршрута в двух режимам: в виде графической схемы и  в виде линейного списка этапов.

При выборе точки действия (этапа) справа отображаются ее настройки.

В области Исполнитель указывают вид исполнителя этапа бизнес-процесса и, при необходимости, значение исполнителя.
В случае групповой точки в данной области отображается  список исполнителей. При этом для элемента с ролевой адресаций становится доступен флаг "Все". Данный флаг указывает, что для каждого пользователя, входящего в эту Роль, будет создана отдельная задача.

В области Настройки этапа указывают длительность этапа (в днях, часах, минутах).

Область Создаваемые на этапе объекты отображает объекты, которые создаются на данном этапе бизнес-процесса (к примеру, документ Событие). Если объект обязателен для создания, то устанавливают флажок в колонке Обязательный. Задача будет закрыта только после создания или выбора хотя бы одного объекта этого типа.

 Если в колонке "Этап основания" указан какой либо этап, при создании документа в качестве его основания можно будет выбрать только ранее введенные на этом этапе документы. Иначе будут доступны все ранее введенные документы.

В области Описание указывается инструкция исполнителю этапа и прочая произвольная информация.
