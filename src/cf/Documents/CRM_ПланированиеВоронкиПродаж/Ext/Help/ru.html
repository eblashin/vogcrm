Документ "Планирование воронки продаж"

Назначение

Документ предназначен для ввода плановых данных по подразделениям, менеджерам или компании в целом для этапов воронок продаж. Соотношение плановых и фактических данных можно посмотреть в обработке «Воронка продаж».

Возможны два варианта планирования воронки продаж:
• Сравнение с эталоном;
• По количеству задач.
