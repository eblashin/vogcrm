Документ "Телемаркетинг"

Назначение

Документ служит для формирования списка клиентов и выполнения массового обзвона по однотипным вопросам. Он позволяет производить телемаркетинг различных групп клиентов с регистрацией состоявшихся контактов, планирование контактов на будущее, проведение опросов (анкетирование).

На форме документа Телемаркетинг располагаются поля для указания номера документа, начала и окончания проведения телемаркетинга, ввода его темы. При указании даты начала и окончания проведения Телемаркетинга (в том числе при установленном флажке На весь день) в календаре пользователя, указанного в реквизите Ответственный, будет зарезервировано время на проведение обзвона.
Поле Маркетинговая кампания предназначено для указания акции, по которой проводится телемаркетинг. Данное поле заполняется значением из справочника Маркетинговые кампании (например, Обзвон клиентов по текущей клиентской базе для кросс-продаж).

В колонке Телефон табличной части закладки Участники документа Телемаркетинг отображается основной телефон клиента (если он указан в карточке клиента). При необходимости, можно выбрать другой телефон из списка доступных в карточке клиента и/или контактного лица  клиента.
Также в табличной части данной закладки находится колонка для возможности создания новой анкеты и интереса или выбора уже имеющейся анкеты т интереса по клиенту.
В документе Телемаркетинг указывается список клиентов, по которым необходимо произвести телефонный обзвон. Список может быть заполнен автоматически при помощи кнопки Заполнить. Возможно заполнение по сегменту клиентов либо по результатам отчета.

На закладке «Сценарий» выводится сценарий, заданный для телемаркетинга на закладке «Реквизиты», а также информация о маркетинговой кампании, количестве активных Интересов текущего абонента, анкетах. В случае отсутствия активных Интересов, анкет, для клиента можно создать Интерес, анкету (по соответствующим гиперссылкам «Создать Интерес», «Создать анкету») во время или по результатам обзвона.

В документе Телемаркетинг может быть указана важность данного телемаркетинга: высокая, обычная или низкая. Также может быть указано состояние телемаркетинга: Запланировано, Завершено, Отменено, В работе, Не обработано или Отложено.
В поле Тема указывается тема обзвона.
Проведенные телефонные разговоры отмечаются флажком в колонке Обработан в табличной части закладки Участники. Если дозвониться не удалось, то в соответствующей строке ставится флажок Не дозвонились.
Под таблицей участников выводится информация о проценте обработанных и необработанных звонков.
