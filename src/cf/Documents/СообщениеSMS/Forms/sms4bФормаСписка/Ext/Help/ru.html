Предназначен для отображения списка документов Сообщения SMS.

Открывается по открывается по команде Сообщения SMS раздела Сервис SMS4B.

В левой части формы отображается дерево состояний сообщений, при выборе строки в котором на список сообщений накладывается отбор по выбранному состоянию.

В списке сообщений выводится:

  • Номер - присваивается автоматически;
  • Дата - дата создания документа;
  • Тип сообщения - входящее или исходящее сообщение;
  • Отправитель - отправитель сообщения;
  • Состояние - состояние сообщение;
  • Статус - текстовое представление состояние сообщения и всех адресатов.
