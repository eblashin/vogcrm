Предназначен для выбора документа Сообщения SMS.

Открывается по кнопке  после выбора типа данных Сообщения SMS (например, при заполнении поля Предмет в каком-либо документе взаимодействий).

В списке выводится:

  • Номер - присваивается автоматически;
  • Дата - дата создания документа;
  • Тип сообщения - входящее или исходящее сообщение;
  • Отправитель - отправитель сообщения;
  • Состояние - состояние сообщение;
  • Статус - текстовое представление состояние сообщения и всех адресатов.

Выбор документа

  • Выделите документ, нажмите Выбрать.
  • Также можно использовать двойной щелчок мыши.

Отбор по периоду

  • С помощью соответствующей команды меню Еще можно Установить период.
