Документ "Сделка"

Документ предназначен для отражения проведения всего цикла продажи клиенту товаров/услуг.

Переход к списку документов производится из раздела "Работа с клиентами" / Панель навигации / «Сделки с клиентами».

В документе необходимо заполнить следующие реквизиты:

  • Проект. Шаблон сделки
  • Важность. Важность продажи.
  • Состояние. Определяется состояние продажи клиенту. В дальнейшем по этому признаку можно будет анализировать причины отказа клиента от покупки товаров/услуг.
  • Начало, окончание. Рассчитываемое или запланированное время начала/окончания всего цикла продажи.
  • Организация. Организация, от которой будет производится выставление счета на оплату клиенту.
  • Контрагент. Контрагент, для которого назначается встреча. Выбирается из справочника "Контрагенты".
  • Договор. Договор с клиентом.
  • Контактное лицо. Контактное лицо контрагента по сделке.
  • Тема. Краткое содержание сделки.
  • Подразделение. Структурная единица организации, к которой относится сделка.
  • Описание. Дополнительные особенности продажи клиенту.
  • Этапы сделки, табличная часть:
    • Этап. Последовательность действий для продажи контрагенту товаров/услуг.
    • Дата начала, срок. Период действия этапа.
    • Ответственный. Ответственный сотрудник за проведение этапа сделки.
    • Виды создаваемых документов. Наименование создаваемых документов для этапа сделки.
